A note taking web application.
- WPA
- should be able to add tile or block, and popup window to add or edit text in the block.
- Text gets saved automatically upon exit or closing the popup window.
- All notes, order of the notes, every setting should be stored in the web browser.
- Ability to export and import notes.
- Heading of the notes, should be `Heading: Note Content` - starts with header, if colon is present.
- Add new tile or block using a plus button below the tile or block.
- Ability to drag and move tile or block up and down, using a three dots or lines on the left.
- Ability to delete the tile or block, upon left swipe.
- Ability to make the tile or block active, upone right swipe.
- Keep the formatting - compact and cozy.
- Add redo and undo buttons on the top, should be greyed out if non avaliable. and should keep 40 redo and undo in memory.
- add dark and light theme, and toggle button.
- add proper footer and header.
- beside the header, keep the buttons for theme, redo, undo, export, import, add button.
